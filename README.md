## Installation
1. Clone the repository
  ```bash
  git clone https://github.com/EldritchTiminy/odin-restaurant.git
  cd odin-restaurant
  ```

2. Install dependencies
  ```bash
  npm install
  ```

3. Start the developement server
  ```bash
  npm run dev
  ```

  ## Usage
  - Easily switch between pages using the navigation bar at the top of the page.
